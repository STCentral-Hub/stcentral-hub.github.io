Welcome to STC Hub

STC Hub is a collection of resources hosted on ST Central.

For those that have been with the ST community for a while you know there are vast amount of player built spreadsheets floating around. ST Central is host to many of these spreadsheets and other tables of data that have outgrown what should be done in those formats. Specifically, loading times have become a serious issue.

STC Hub has been created to replace those bulky spreadsheets with lean statically generated html which will also enable much richer content.

Please join us on the ST Central Discord!
